# 알찬방학

우리 아이 방학 계획표 생성 및 관리 서비스

## 프로젝트 소개

알찬방학은 학부모와 학생이 방학 기간 동안 할 일을 계획하고 매일 체크하며 달성률을 확인할 수 있는 서비스입니다.

복잡한 계획표 대신 쉽고 직관적인 UI를 제공하며, 목표 설정부터 요일별 할 일 관리, 일일 완료 체크, 달성률 확인까지 한 번에 관리할 수 있습니다.

향후 AI 기반 계획표 자동 생성 기능을 추가하여 더욱 편리한 방학 관리 경험을 제공할 예정입니다.

## 구현된 기능

### MVP 1단계 ✅

#### 방학 정보 설정
- 아이 이름, 방학 시작일/종료일, 목표 입력
- 수정 저장 전 내용 확인 모달 / 변경사항 없을 때 별도 안내
- 저장된 정보 미리보기 카드

#### 할 일 관리 (홈 탭 카드)
- **반복 할 일** 탭: 월~일 요일별 할 일 추가 / 수정 / 삭제, 시간 설정 포함
  - 반복 할 일을 평일/주말/매일 등 여러 요일에 한 번에 추가
- **특별 할 일** 탭: 특정 날짜에만 표시되는 1회성 일정
  - 날짜 선택 + 선택적 시간 + 텍스트 입력
  - 중요 표기(★) 설정 가능 — 캘린더 전 뷰에서 강조 표시
  - 수정 / 삭제 지원

#### 캘린더 (3가지 보기)
- **리스트 보기**: 방학 기간 전체 날짜 목록, 오늘 자동 스크롤 및 강조
- **달력 보기**: 월간 달력 그리드, 중요 일정 ★ 표기, 마일스톤 선물 아이콘
- **다이어리 보기**: 날짜별 줄노트 페이지 UI
  - 좌우 버튼 + 스와이프 제스처로 날짜 이동, 3D 페이지 플립 애니메이션
  - 5가지 컬러 테마 선택 (주황 / 파랑 / 핑크 / 민트 / 레인보우)
  - 글자 크기 조절 버튼 (100%~157%, LocalStorage 저장)

#### 완료 체크
- 오늘 날짜에만 체크 가능 (날짜 제한)
- 완료 상태 LocalStorage 저장, 모든 뷰 간 동기화

#### 메모
- 자유 텍스트 메모 (고민·아이디어 기록용), 입력 즉시 자동 저장

---

### MVP 2단계 ✅

#### 스티커판 (도장판)
- 방학 전체 기간을 5열 그리드로 표시
- 당일 할 일 전부 완료 시 도장 자동 찍힘 (주말·할 일 없는 날 자동 도장)
- 도장 안 아이콘: 마일스톤=선물, 응원=하트, 기본=별

#### 마일스톤 테마 해금
- 방학 7 / 14 / 21 / 30일차 전부 완료 시 다이어리 테마 해금
- 해금 전 회색 예고 → 해금 시 컬러 + 팝업 안내

#### 연속 달성 스트릭
- 현재 연속 달성일 + 최고 기록 배너 (캘린더·스티커판)

#### 엄마아빠의 응원
- 날짜별 응원 메시지 / 선물 등록
- 메시지: 당일에만 공개 / 선물: 즉시 공개
- 스티커판 하트 탭하면 팝업으로 내용 확인

#### 홈화면 PIN 잠금
- 부모 전용 설정 보호용 4자리 PIN 잠금
- PIN 분실 시 자물쇠 아이콘 3초 길게 누르면 비상 초기화 가능

#### 입력 필터링
- 모든 텍스트 입력 필드에 욕설 실시간 필터링 (`utils/filter.ts`에서 단어 목록 관리)
