# 알찬방학

우리 아이 방학 계획표 생성 및 관리 서비스

## 프로젝트 소개

알찬방학은 학부모와 학생이 방학 기간 동안 할 일을 계획하고 매일 체크하며 달성률을 확인할 수 있는 서비스입니다.

복잡한 계획표 대신 쉽고 직관적인 UI를 제공하며, 목표 설정부터 요일별 할 일 관리, 일일 완료 체크, 달성률 확인까지 한 번에 관리할 수 있습니다.

향후 AI 기반 계획표 자동 생성 기능을 추가하여 더욱 편리한 방학 관리 경험을 제공할 예정입니다.

## 구현된 기능 (MVP 1단계)

### 방학 정보 설정
- 아이 이름, 방학 시작일/종료일, 목표 입력
- 수정 저장 전 내용 확인 모달
- 저장된 정보 미리보기 카드

### 할 일 관리 (홈 탭 카드)
- **반복 할 일** 탭: 월~일 요일별 할 일 추가 / 수정 / 삭제, 시간 설정 포함
  - 반복 할 일을 평일/주말/매일 등 여러 요일에 한 번에 추가
- **특별 할 일** 탭: 특정 날짜에만 표시되는 1회성 일정
  - 날짜 선택 + 선택적 시간 + 텍스트 입력
  - 중요 표기(★) 설정 가능 — 캘린더 전 뷰에서 강조 표시
  - 수정 / 삭제 지원

### 캘린더 (3가지 보기)
- **리스트 보기**: 방학 기간 전체 날짜 목록
  - 오늘 날짜 자동 스크롤 및 강조 뱃지
  - 반복 + 특별 할 일 통합 표시, 진행 게이지
- **달력 보기**: 월간 달력 그리드 (월요일 시작)
  - 오늘 날짜 자동 스크롤 + 주황 테두리 원 표시
  - 중요 특별 할 일 있는 날 ★ 표기
  - 완료 날짜 오렌지 원 표기, 날짜 클릭 → 팝업 체크
- **다이어리 보기**: 날짜별 줄노트 페이지 UI
  - 좌우 버튼 + 스와이프 제스처로 날짜 이동
  - 3D 페이지 플립 애니메이션
  - 기기 화면 높이에 맞춘 고정 크기 페이지

### 완료 체크
- 체크박스 또는 텍스트 클릭으로 완료 토글
- 반복 할 일: 인덱스 기반 추적 / 특별 할 일: ID 기반 추적 (독립 관리)
- 완료 상태 LocalStorage 저장, 모든 뷰 간 동기화
- 완료 시 취소선 + 카드/페이지 배경 변경

### 메모
- 자유 텍스트 메모 (고민·아이디어 기록용)
- 입력 즉시 자동 저장, 새로고침 후에도 유지

### UX
- 캘린더 탭 선택 상태 유지 (앱 재진입 시 마지막 탭 복원)
- 상단 제목 + 탭바 sticky 고정 (스크롤 시 항상 표시)
- 앱 진입 시 데이터 있으면 캘린더로 자동 이동
