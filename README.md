# 알찬방학

우리 아이 방학 계획표 생성 및 관리 서비스

## 프로젝트 소개

알찬방학은 학부모와 학생이 방학 기간 동안 학습, 독서, 운동, 체험활동 등을 계획하고 관리할 수 있도록 돕는 서비스입니다.

복잡한 계획표 대신 쉽고 직관적인 UI를 제공하며, 목표 설정부터 일일 체크, 달성률 확인까지 한 번에 관리할 수 있습니다.

향후 AI 기반 계획표 자동 생성 기능을 추가하여 더욱 편리한 방학 관리 경험을 제공할 예정입니다.

## 주요 기능 (MVP)

### 1. 방학 정보 설정

- 방학 시작일
- 방학 종료일
- 학생 이름 입력

### 2. 목표 설정

- 공부 목표
- 독서 목표
- 운동 목표
- 기타 목표

### 3. 계획표 생성

- 날짜별 일정 생성
- 주간 계획 확인
- 월간 계획 확인

### 4. 일일 체크

- 완료 체크
- 진행률 확인
- 연속 달성 기록

### 5. 통계

- 전체 달성률
- 목표별 달성 현황
- 완료한 활동 수

## 향후 개발 예정

### AI 계획표 생성

예시

> 초4 여학생
> 영어 30분
> 독서 20분
> 수학 복습 위주
> 주말은 휴식

AI가 입력 내용을 분석하여 방학 계획표를 자동 생성
