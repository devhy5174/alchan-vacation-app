# Changelog

## [0.5.0] - 2026-06-23

### Added

**PIN 비상 초기화**
- 잠금 화면 자물쇠 아이콘 3초 길게 누르면 원형 게이지 표시 후 PIN 초기화 확인 모달 출력
- 초기화 시 PIN 삭제 + 즉시 홈 진입, 이후 새 PIN 재설정 가능

**다이어리 글자 크기 조절**
- 하단 네비게이션 중앙(페이지 표시 아래)에 `− 100% +` 버튼 추가
- 100% ~ 157%(+8단계)까지 조절, 줄 간격도 비례해서 확대
- 설정값 LocalStorage 저장 — 앱 재실행 후에도 유지
- 테마 accentColor 기반 pill 스타일 버튼

**욕설 필터링**
- 모든 텍스트 입력 필드에 실시간 욕설 필터 적용 (입력 즉시 `***` 치환)
- `src/utils/filter.ts` 한 파일에서 단어 목록 관리

**방학 정보 변경사항 없음 안내**
- 수정 후 저장 시 기존 데이터와 동일하면 확인 모달 없이 "변경사항이 없어요" 팝업 표시

### Fixed

**Android 소프트 키보드 Enter 키 미동작**
- `onKeyDown` → `onKeyUp` 으로 변경 (MemoSection, ParentRewardSection)
- Android IME에서 keydown 이벤트가 `Unidentified`로 전달되는 이슈 해결

## [0.4.0] - 2026-06-21

### Added

**스티커판 (도장판)**
- 방학 전체 기간을 5열 그리드로 표시, 당일 할 일 전부 완료 시 도장 자동 찍힘
- 주말·할 일 없는 날 자동 도장
- 도장 아이콘 구분: 마일스톤=선물, 응원=하트, 기본=별
- 캘린더 페이지 상단 버튼으로 접근

**마일스톤 다이어리 테마 해금**
- 방학 7 / 14 / 21 / 30일차 전부 완료 시 다이어리 컬러 테마 순차 해금
- 스티커판: 해금 전 회색 선물 아이콘 예고 → 해금 시 테마 컬러 + 팝업 안내
- 달력 탭: 마일스톤 날짜에 선물 아이콘 표시 (해금 전 회색, 후 컬러)

**연속 달성 스트릭 배너**
- 현재 연속 달성일 + 최고 기록 표시 (캘린더·스티커판)

**엄마아빠의 응원**
- 홈 응원 탭에서 날짜별 응원 메시지 / 선물 등록
- 선물: 즉시 공개 (항상 하트 표시) / 메시지: 당일에만 공개
- 수정 / 삭제 시 확인 모달
- 스티커판 하트 탭하면 팝업으로 내용 확인

**홈화면 PIN 잠금**
- 부모 전용 설정 보호용 4자리 PIN 잠금
- 홈 진입 시 PIN 확인, 캘린더 이동 시 자동 초기화
- PIN 설정 / 변경 / 해제 지원

## [0.3.0] - 2026-06-19

### Added

**다이어리 보기 탭**
- 캘린더 페이지에 3번째 탭 '다이어리' 추가 (리스트 / 달력 / 다이어리)
- 날짜별 줄노트 스타일 페이지 UI (크림색 배경 + repeating-linear-gradient 줄)
- 좌우 버튼 + 터치 스와이프로 날짜 이동
- CSS perspective + rotateY 기반 3D 페이지 플립 애니메이션 (exit 0.2s / enter 0.22s)
- 기기 화면 높이에 맞춘 고정 높이 카드 (`clamp(200px, calc(100svh - 330px), 480px)`)
- 오늘 날짜 "오늘의 방학 페이지" 헤더 + 주황 강조
- 이전/다음 버튼에 실제 날짜 표시, 가운데 "오늘로" 버튼

**특별 할 일 (특정 날짜 1회성 일정)**
- 홈 할 일 카드를 '반복 할 일 / 특별 할 일' 탭으로 분리
- 날짜 선택 + 선택적 시간 + 텍스트 입력으로 특정 날 할 일 추가
- 중요 표기(★) 토글: 추가/수정 폼 모두 지원
- 수정(인라인) / 삭제 지원
- LocalStorage 독립 키 저장 (`alchan_specific_tasks`, `alchan_specific_completion`)
- ID 기반 완료 추적 — 삭제 시 인덱스 충돌 없음
- 리스트 / 달력 상세모달 / 다이어리 전 뷰에서 반복 할 일과 통합 표시

**중요 할 일 강조**
- 리스트·다이어리·상세모달: 중요 특별 할 일에 ★ 아이콘 + `text-orange-500 font-medium`
- 달력 보기: 중요 미완료 특별 할 일 있는 날짜 셀에 진노란 ★ 표기

**캘린더 달력 보기 개선**
- 오늘 날짜 셀 주황 테두리 원 표시
- 진입 시 오늘 날짜로 자동 스크롤
- 잔여 할 일 수 숫자 제거 (중요 ★만 표기)

**UX**
- 캘린더 탭 선택 상태를 localStorage에 저장 — 재진입 시 마지막 탭 복원
- 캘린더 페이지 상단 제목 + 탭바 sticky 고정 (스크롤 시 항상 표시)

## [0.2.0] - 2026-06-19

### Added

**캘린더 뷰 개선**
- 리스트 보기: 오늘 날짜 카드 자동 스크롤 + "오늘" 뱃지 표시
- 리스트 보기: 할 일 텍스트 클릭해도 완료 토글 (체크박스 클릭 외)
- 달력 보기: 완료된 날짜 셀 배경색 변경 (bg-orange-50)
- 달력 보기: 남은 할 일 수 셀에 표시, 완료 시 오렌지 원으로 표기
- DayDetailModal: 중앙 모달 팝업, 게이지 아래 확인 버튼 추가

**UX 개선**
- 앱 진입 시 캘린더 데이터 있으면 /calendar로 자동 이동
- 방학 정보 폼 아코디언 (저장 후 자동으로 접힘)
- 정보 수정 저장 시 ConfirmModal로 내용 확인 후 저장
- AlertModal: 저장하기/캘린더만들기 후 결과 알림 (확인 버튼 포함)

**시간 선택 피커**
- TimePickerModal: 오전/오후 토글, 시·분 직접 입력 + ▲▼ 버튼
- DayScheduleEditor, RepeatTaskAdder에 시계 아이콘 버튼으로 피커 연결
- "시간 없음" 옵션으로 시간 제거 가능

**메모 기능**
- 요일별 할 일 위에 자유 메모 섹션 추가
- 입력 즉시 LocalStorage 자동 저장, 새로고침 후에도 유지
- 전체 삭제 버튼 제공

### Fixed
- 날짜/이름 수정 후 저장 시 weeklySchedule 초기화되는 버그 수정

## [0.1.0] - 2026-06-18

### Added

**기초 설정**
- React + TypeScript + Vite + Tailwind CSS 프로젝트 초기화
- react-router-dom 기반 라우팅 구조 (/, /calendar)
- 하단 고정 BottomNavigation (홈/캘린더 탭)
- Button, Input 공통 컴포넌트

**방학 정보 입력**
- 아이 이름, 방학 시작일/종료일, 목표 입력 및 LocalStorage 저장
- VacationPreviewCard: 저장된 방학 정보 미리보기

**요일별 할 일 입력**
- 월~일 요일별 할 일 추가/수정/삭제
- 할 일마다 선택적 시간 입력 (HH:MM)
- RepeatTaskAdder: 반복 할 일을 평일/주말/매일 등 여러 요일에 한 번에 추가 (중복 방지)
- IME 한국어 입력 버그 수정 (onKeyDown → onKeyUp)

**캘린더 페이지**
- 리스트 보기: 방학 기간 날짜 목록, 요일별 할 일 및 시간 표시
- 달력 보기: 월간 달력 그리드 (월요일 시작), 방학 기간 강조
- 리스트/달력 탭 전환

**할 일 완료 체크**
- 날짜별 체크박스 완료 토글, LocalStorage 영속화
- 날짜 카드 진행 게이지 + 완료/전체 카운트
- 완료 시 카드 배경색 변경 (bg-orange-50)
- 달력 셀 클릭 → DayDetailModal 팝업으로 체크 가능
